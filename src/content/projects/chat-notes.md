An intelligent, mobile-first note-taking app. One frictionless chat box for all types of notes — tasks, ideas, meetings, journal entries, reading notes. AI categorizes everything automatically using GPT-4o.

## Features

- **Single input interface** — one chat box for all types of notes, zero friction
- **AI-powered organization** — automatic categorization with 85%+ accuracy
- **Rich metadata extraction** — intelligently extracts due dates, meeting times, links, and more
- **Type-specific views** — dedicated interfaces for tasks, ideas, meetings, journal, and reading
- **Full-text search** — instant search across all notes with live results
- **Mobile-first** — optimized for mobile with PWA capabilities
- **Real-time cost tracking** — monitor AI usage costs with granular per-request tracking
